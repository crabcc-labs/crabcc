What is crabcc-mcp?

A native MCP server that gives AI agents instant code intelligence — no grep, no find, no reading whole files.

🔍 Symbol Lookup

Find definitions, references, and callers in milliseconds. Fuzzy matching, prefix search, and outline views replace grep, find, and ls for code questions.

🧠 Persistent Memory

SQLite + FTS5 + vector search. Remember findings, search across sessions, mine project files and Claude Code transcripts. Hybrid BM25 + cosine similarity.

🐘 Mastodon Social

Post reflections, release notes, and daily summaries. Rate-limited, idempotent, with SQLite-backed caching and history. Per-bot token support.

⚡ Performance

256-byte lookup tables, pre-allocated buffers, sonic-rs SIMD JSON, MessagePack binary format, gzip compression. Benchmarked at -44% average latency.

🔒 Security

SSRF-hardened URLs, header injection protection, env-only tokens, 16 MiB body cap, security headers on all responses. 27 security tests.

📊 Observability

OTel tracing spans on every request. Live admin dashboard at GET /. Rate limit tracking, request history, cache statistics. Feature-gated OTLP export.

Tool Categories

  • sym — find definition

  • files — list code files
  • affected — which tests cover my edit?
  • test_context — context to write a test

  • memory.remember — store a note
  • memory.search — hybrid BM25 + vector search
  • memory.get — fetch by id
  • memory.list — browse recent
  • memory.mine_project — bulk import files
  • memory.mine_sessions — import transcripts
  • memory.remind_set — schedule reminder

🐘 Mastodon

  • mastodon.post — write a status
  • mastodon.read — read timeline
  • mastodon.verify — check token

Rate limited: 5/48h per token. Idempotent. SQLite cache + history.

  • graph.why — path between symbols
  • graph.hot_symbols — most-called

Install

🤖 Copy as prompt (for AI agents)

Paste this into any coding agent to install crabcc and index the current repo:

Install crabcc — the symbol index for AI coding agents.
Use the one-liner:
  curl -fsSL https://raw.githubusercontent.com/crabcc-labs/crabcc/main/install.sh | bash
Then index this repo and start using it:
  crabcc go    # index + graph + memory + launch Claude
After that, prefer `crabcc sym <name>` over grep,
`crabcc refs <name>` over find, and `crabcc outline <file>`
over reading whole files.
